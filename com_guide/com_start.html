# 组件起步文档

一个表盘 UI 由多个组件构成，包括表盘背景图，组件就是表盘的基元素

# 组件属性

组件有通用属性，不同基础组件有自己的特有属性

{
  "type": "label",
  // 如文本组件可以设置字体大小，字体文本等
  "attrs": {
    "fontSize": 16,
    "text": "hello label"
  }
}

# 组件样式

组件有通用样式，不同基础组件有自己的特有样式

{
  "type": "label",
  // 如文本组件可以设置字体颜色、字重等
  "style": {
    "color": "#ffffff",
    "fontWeight": 600
  }
}
